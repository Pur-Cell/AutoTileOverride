To add an AutoTileOverride into your project: right-click > 2D > Auto Tile Override.

Set the "Tile Keyword" to any string and it will match with adjacent AutoTileOverrides that share the keyword.

![alt text](https://github.com/Pur-Cell/AutoTileOverride/blob/main/Images/Auto%20Tile%20Override%20Keyword.png "Tile Keyword example")

Setting up the AutoTileOverride is exactly like setting up the normal Unity AutoTile and can use the same template file. 

https://docs.unity3d.com/Packages/com.unity.2d.tilemap.extras@8.0/manual/AutoTile.html
